Good Librarian!

According to "Gazelle Table Logic," Editions of the same Source (for example, two translations of Plato's Republic) should be listed under the same Gazelle Table Heading. So before you upload anything, 1) Type the Source title into autocomplete. 2) If it exists, you will go to the Source's Upload page; 3) Then add the new Edition from that Source's Upload Page. If the Source doesn't come up, simply type the Title into the input field to upload a "New Source."


More Advanced: 4) If, for example, an audiobook and ebook have the same edition (or translation), select the existing EDITION from the Edition drop-down menu after autocompleting the Source Title. This way, Diamond Sutra Trans. Red Pine will list a PDF and mp3 under the same Edition heading when the Gazelle Table is rendered.


Ebooks, Audiobooks, Classical Music, and Documentaries are allowed.


Thus the Graph Logic is Source->Edition->Torrent.


When you add a New Author, be sure to click the "Create Author" button, after providing the name in APA Format (Last Name, Initials.) If the Author already exists on the site, find the name in the dropdown bubble. If you do not autocomplete or Create an Author, no Author will be established. If the APA format of an Author already exists, but it is not the same as your Source Author, put that author's genre in (Paranthesis) - e.g., Smith, J. (Alchemist)

How to Seed:

Please only upload if you can seed willingly, very preferrably with BiglyBT (using their WebTorrent plugin) or WebTorrent Desktop. In the Upload Prompt, you must upload the media you wish to distribute, not the ".torrent" file. An infoHash will be generated using WebTorrent in the Browser as soon as you upload your media. After WebTorrent generates the infoHash, please download the MagnetURI or Torrent File, and seed from a Desktop client or seedbox for as long as you can.
